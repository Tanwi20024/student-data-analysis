# 📊 Student Performance Data Analysis

A beginner-friendly Data Analysis project built using **Python**, **Pandas**, and **Matplotlib** to analyze student performance data.

---

## 🎯 Objective
To analyze student performance data by calculating average scores and creating visualizations to derive meaningful insights.

---

## 🛠️ Libraries Used
| Library | Purpose |
|---------|---------|
| Pandas | Loading and analyzing data |
| Matplotlib | Creating charts and graphs |
| Seaborn | Creating heatmaps |
| NumPy | Numerical calculations |

---

## 📂 Project Structure
- **Step 1** - Import all necessary libraries
- **Step 2** - Create and load the dataset
- **Step 3** - Explore the dataset
- **Step 4** - Calculate averages
- **Step 5** - Bar Chart – Average score by subject
- **Step 6** - Scatter Plot – Study hours vs Average score
- **Step 7** - Heatmap – Correlation between all variables
- **Step 8** - Key insights and observations

---

## 📊 Visualizations
- 📊 **Bar Chart** – Compares average scores across Math, Science and English
- 📈 **Scatter Plot** – Shows relationship between study hours and average score
- 🔥 **Heatmap** – Shows correlation between all numeric variables

---

## 🔍 Key Insights
- **English** is the strongest subject with **78.93 average**
- **Study Hours** have a **0.96 correlation** with average score
- **Attendance** has a **0.95 correlation** with average score
- **Top Performer** is Charlie with an average score of **91.67**
- **Female students** averaged **84.08** vs Male students at **71.57**
